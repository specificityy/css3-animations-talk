Sprite Animations

							.ken {
  height: 110px;
  width: 70px;
  background-image: url("/css/img/ken-shoryuken.png");
  animation: shoryuken steps(15, end) normal 2.5s infinite;
}

@keyframes shoryuken {
  from {
    background-position: 0 0;
  }
  to {
    background-position: -1054px 0;
  }
}
The second parameter of step() defines the point at which the action declared in @keyframes will start.
  • End: is default, starts at 1st part of the img.
  • Start: animation starts at 2nd part of img.
  • This is how you see sprites most of the time.

Typing sprite animation. 

							.typing-animation span {
  width: 0;
  background: #222;
  border-left: .1em solid #222;
  animation: typing 8s steps(24, end), blink-caret 1s step-end infinite;
}

@keyframes typing {
  from { width: 100% }
  to { width:0 }
}

@keyframes blink-caret {
  from, to { border-color: transparent }
  50% { border-color: #fff }
}

SVG Path Drawing

  • Dasharray: lets you specify the length of the rendered part of the line, then the length of the gap.
  • Dashoffset: lets you change where the dasharray starts.
var path = document.querySelector('.talk path');
var length = path.getTotalLength();

// Clear any previous transition
path.style.transition = 'none';

// Set up the starting positions
path.style.strokeDasharray = length + ' ' + length;
path.style.strokeDashoffset = length;

// Trigger a layout so styles are calculated & the browser
// picks up the starting position before animating
path.getBoundingClientRect();

// Define our transition
path.style.transition =	'stroke-dashoffset 2s ease-in-out';

// Draw!
path.style.strokeDashoffset = '0';
